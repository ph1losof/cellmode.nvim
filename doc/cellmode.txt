*cellmode.txt*  cellmode.nvim - fast table-mode editing for Neovim

INTRODUCTION                                          *cellmode*

cellmode is an adapter-driven Neovim plugin for CSV/TSV-style
table workflows. It renders a fast viewport-oriented table view while
keeping structural operations explicit.

COMMAND                                               *:Cellmode*

Use the main command:

  :Cellmode open <path> <format> [--adapter <cmd> [args...]]
  :Cellmode op set-cell <segment> <row> <col> <value>
  :Cellmode op insert-row <segment> <row> <value1,value2,...>
  :Cellmode op delete-row <segment> <row>
  :Cellmode save <path> [format]
  :Cellmode status

SETUP                                                 *cellmode-setup*

lua << EOF
require("cellmode").setup({
  adapters = {
    csv = { command = { "python3", "/path/to/adapters/cellmode_tir_csv_adapter.py" } },
    tsv = { command = { "python3", "/path/to/adapters/cellmode_tir_csv_adapter.py", "--delimiter", "\t" } },
  },
})
EOF

Defaults automatically register CSV/TSV adapters through the bundled
script when adapters are not provided.
